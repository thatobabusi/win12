# Credits

**English** · [简体中文](CREDITS_zh.md) · [Setswana](CREDITS_tn.md)

This project is a fork of the open-source **Windows 12 Online** (Windows 12 网页版)
web project. The original design and the overwhelming majority of this codebase
were created by the original authors and community credited below. This fork
exists for **educational purposes** and builds on their work — full credit for
the foundation belongs to them.

## Original project

- **Name:** Windows 12 Online (Windows 12 网页版)
- **Original repository:** <https://github.com/win12-online/win12> *(no longer publicly available at the time of writing)*
- **Translations repository:** <https://github.com/win12-online/win12-locales>
- **Original theme assets:** <https://github.com/tjy-gitnub/win12-theme>
- **Project site:** <https://win12.tech>

> As of mid-2026 the upstream repository and its site appear to be offline or set
> to private. This is noted without any complaint — we're grateful the work was
> shared openly, and we simply want to preserve proper attribution. If any
> original author would like this credit adjusted, or asks that this fork be
> taken down, we will honour that.
