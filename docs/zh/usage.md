# 使用

> 🌐 English: [../en/usage.md](../en/usage.md)

win12 桌面运行起来后的快速上手。

---

## 开机与登录

1. 打开站点根地址——开机界面会自动运行（约 2 秒）。
2. 开机过程中按 **F2** 可进入 BIOS/SETUP 界面（可选）。
3. 在登录界面，如需要可选择语言（右下角列表），然后点击**登录**。覆盖层淡出，桌面出现。
4. 开发时如需跳过登录，追加 `?skip_login=1`。

---

## 桌面基础

| 操作 | 方法 |
|------|------|
| 打开应用 | 双击桌面图标，或使用任务栏 / 开始菜单 |
| 右键菜单 | 在空白桌面右键 → 刷新、切换主题、个性化等 |
| 移动窗口 | 拖动其标题栏 |
| 运行中的应用 | 显示在居中的任务栏 Dock 中（仅在有应用打开时可见） |
| 切换语言 | 设置应用，或登录界面的语言列表 |

---

## 应用

内置应用包括：设置、文件资源管理器、Microsoft Edge、计算器、记事本、终端、商店、相机、
白板、Defender、Word、Copilot、图片查看器、媒体播放器、代码编辑器，以及游戏（如扫雷）。
具体可用性会随分支演进而变化。

---

## 提示

- **看起来异常（图标缺失、登录透明）？** 几乎总是 Service Worker 缓存过期。刷新两次，
  或使用 `?develop=1` 加载。见[配置](configuration.md#service-worker)。
- **正在改代码？** 使用 `?develop=1`，避免 Service Worker 给你返回缓存文件。
- **重置状态？** 应用状态保存在 `localStorage` 中；清除站点数据会重置桌面、语言与主题。
